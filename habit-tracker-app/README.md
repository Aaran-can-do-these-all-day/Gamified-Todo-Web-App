# Habit Tracker App

## Overview
The Habit Tracker App is designed to help users build and maintain healthy habits through effective tracking and visualization. The app features a monthly calendar heatmap, a daily checklist, and customizable settings to enhance user experience.

## Features
- **Month View**: 
  - Calendar-style heatmap for tracking habit completion.
  - Navigation to switch between months.
  - Monthly summary including total tasks completed, best streak, most active day, and number of active days.
  - Habit filtering options.

- **Agenda View**: 
  - Daily checklist for today's habits.
  - Ability to tick off habits with a single tap.
  - Display of reminders and scheduled times for habits.
  - Option to add daily notes.

- **Settings View**: 
  - Add, edit, or delete habits.
  - Change habit colors.
  - Set reminders and notifications.
  - Toggle between light mode, dark mode, and AMOLED mode.
  - Choose the weekly start day.
  - Backup and restore habit data.
  - Reset progress for a habit or the entire app.

## Additional Features
- Floating “+” button for creating new habits or adding today’s entry.
- Analytics for identifying the weakest and strongest habits, and longest streaks.
- Option to export progress as an image or PDF.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd habit-tracker-app
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage
- Navigate between the Month, Agenda, and Settings views using the buttons at the top of the page.
- In the Month View, visualize your habits with a heatmap and access your monthly summary.
- In the Agenda View, manage your daily habits and add notes.
- Customize your habits and app settings in the Settings View.
